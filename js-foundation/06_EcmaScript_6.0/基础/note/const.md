# const 命令

- const 用于声明不可改变的量(常量)
- const 拥有 let 的所有性质
- const 命令的实质
    - 原始值不可改变
        - 因为原始值一旦改变就需要重新分配内存空间
    - 引用值可以改变
        - 引用值只要不是重新赋值,对其进行改变是可以的,因为没有重新分配内存空间
    ```
    const MAX = 999;
    MAX = 123;// 报错
    ```
    ```
    const obj = {};
    obj.a = 123;
    console.log(obj);
    ```
