# JavaScript 读书笔记

- 变量实际上也是伪类,因为它拥有自身的属性(attributes)

    - 该属性决定了该变量是否可以被修改删除

    - 或者在for-in循环中进行枚举

    - es5中提供了一个特殊的描述符方法来操纵这些属性

- 对象有两种类型

    - 原生的(Native)
        - 原生对象进一步分为
            > 内置对象(数组,日期对象等)

            > 和用户自定义对象

    - 主机的(Host,即在主机环境中定义的)

    
- 主机对象包含所有的DOM对象和windows对象
    - (可以在不同的环境下运行该js代码,如果可以正确运行那么使用的就是Native对象)
