# 泛型的练习 -- 自定义字典类

- 开发一个字典类(Dictionary), 字典中会保存键值对的数据

- 键值对数据的特点
    - Key
        - 可以是任何类型
        - 所有的键类型要相同
        - 不允许重复
    - Value
        - 可以是任何类型
        - 所有的值类型要相同
    - 键和值一一对应

- 字典类中对键值对数据的操作(增删改查+循环计数)
    - 重新设置某个键对应的值,如果不存在则添加
    - 按照键,删除对应的键值对
    - 判断某个键是否存在
    - 循环每一个键值对
    - 得到当前键值对的数量
